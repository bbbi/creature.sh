# Supraciliary

Professional portfolio with refined dark theme.

## Structure

```
supraciliary-portfolio/
├── index.html      # Semantic HTML structure
├── styles.css      # Premium color system, 400+ lines
├── script.js       # Subtle interactions
└── README.md       # This file
```

## View

```bash
open index.html
```

## Features

- **Refined Color Palette** - Carefully tuned indigo accent system
- **Typography Hierarchy** - Inter + JetBrains Mono
- **Smooth Animations** - Scroll-triggered fades, parallax
- **Hover Effects** - Subtle glows and transforms
- **Accessible** - Respects reduced motion preferences
- **Responsive** - Mobile-first grid layout

## Deploy

**Netlify:** Drag folder to https://app.netlify.com/drop

**Vercel:** `vercel` in directory

**GitHub Pages:** Push repo, enable Pages
